# 📚 Letrinhas – App Educativo para Dislexia

> Aplicativo Flutter desenvolvido como Trabalho de Conclusão de Curso (TCC), com foco em acessibilidade e apoio ao processo de alfabetização de crianças com dislexia.

[![Flutter](https://img.shields.io/badge/Flutter-3.0+-02569B?logo=flutter)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/Firebase-Enabled-FFCA28?logo=firebase)](https://firebase.google.com)

---

## 🎯 Sobre o Projeto

**Letrinhas** é um aplicativo educativo que auxilia crianças com dislexia por meio de atividades interativas, gamificação e recursos de acessibilidade.

O app foi desenvolvido considerando princípios de design acessível, usabilidade infantil e apoio ao processo de alfabetização.

### Recursos de acessibilidade implementados
- 🔤 Fonte OpenDyslexic
- 🎨 Modo Alto Contraste
- 🔊 Text-to-Speech (TTS) em Português Brasileiro
- 📏 Ajuste de tamanho de texto e ícones
- 🎬 Controle de animações

---

## ✨ Funcionalidades Implementadas

### 🔐 Autenticação
- Login e cadastro de usuários
- Recuperação de senha por e-mail
- Autenticação com Firebase Authentication

### 🎮 Atividades Educativas
O aplicativo possui **5 atividades funcionais**:

1. Associar palavras com imagens  
2. Completar palavras com letras faltantes  
3. Ordenar sílabas corretamente  
4. Leitura e interpretação de frases  
5. Associação de áudio com imagem  

### 🏆 Gamificação
- Sistema de pontuação
- Sistema de níveis
- Barra de progresso
- Registro de atividades concluídas

### 👤 Perfil do Usuário
- Visualização de estatísticas pessoais
- Histórico de atividades realizadas
- Acompanhamento de progresso

### ⚙️ Configurações de Acessibilidade
- Fonte OpenDyslexic (ativar/desativar)
- Modo alto contraste
- Ajuste de tamanho do texto
- Ajuste de tamanho dos ícones
- Controle de animações
- Controle de sons e TTS

## ♿️ Acessibilidade

O aplicativo foi desenvolvido com foco em crianças com dislexia, priorizando:

- Legibilidade
- Redução de sobrecarga cognitiva
- Estímulos visuais controlados
- Feedback auditivo

As configurações de acessibilidade são ajustáveis pelo usuário e salvas automaticamente.
